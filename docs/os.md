# 操作系统

## 进程与线程

- **进程 vs 线程**：资源隔离、切换开销、通信方式（管道、消息队列、共享内存）
- **进程状态**：就绪 → 运行 → 阻塞 → 就绪（五态模型）
- **调度策略**：FCFS、SJF、优先级调度、时间片轮转、多级反馈队列

## 内存管理

- **分页**：页表、TLB 加速、页面置换算法（LRU、LFU、Clock）
- **分段**：逻辑分段与物理映射
- **虚拟内存**：缺页中断、demand paging、swap 交换分区

## 并发与同步

- **互斥锁 / 自旋锁**：适用场景与开销对比
- **信号量**：P/V 操作，用于资源计数与同步
- **死锁**：四个必要条件（互斥、占有等待、不剥夺、循环等待）、预防/避免/检测/解除
- **条件变量**：与互斥锁配合使用

## IO 与文件系统

- **IO 模型**：阻塞、非阻塞、IO 多路复用（select/poll/epoll）、异步 IO
- **epoll**：边缘触发（ET）vs 水平触发（LT），高并发实现原理
- **文件系统**：inode 结构、软链接 vs 硬链接、缓冲区缓存
